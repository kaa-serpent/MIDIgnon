Home Setup Using MIDIgnon

Step 1: Connect the Device

Plug the MIDIgnon into your computer using a USB cable. It will appear as a MIDI device.

Step 2: Install Drivers (if needed)

Most computers detect MIDIgnon automatically. If not, check your OS instructions for USB MIDI devices.

Step 3: Open Your Music Software

Open your DAW, synth, or MIDI host software. In the MIDI input settings, enable MIDIgnon as an input device.
